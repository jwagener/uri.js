## Development
### Requirements

- QUnit
- CoffeeScript
- JSL (JavaScript Lint)
- rake

### Getting started
    # open the test suite
    $ rake test 
    
    # watch coffeescript and build as soon as it's modified
    $ rake watch      

    # build and minify
    $ rake build
